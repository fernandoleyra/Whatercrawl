# Changelog

## [1.0.0] — 2026-06-25

### Added
- First release as a pure Claude Code plugin (no Docker, no running service, no API key required).
- Nine skills: watercrawl-scrape, watercrawl-crawl, watercrawl-extract, watercrawl-search, watercrawl-map, watercrawl-screenshot, watercrawl-links, watercrawl-batch, watercrawl-monitor.
- Ten commands: /watercrawl router + one per skill.
- Screenshot support via optional Playwright installation.
- Page monitoring with local snapshots and change detection.
- Zero-configuration install: no API keys, no Docker, no config files.
