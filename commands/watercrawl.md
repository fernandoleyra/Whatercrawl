# Watercrawl

Watercrawl gives you web scraping, crawling, and extraction natively in Claude Code — no service, no API key, no Docker.

## Available commands

| Command | What it does |
|---|---|
| `/watercrawl-scrape <url>` | Fetch a single page as Markdown |
| `/watercrawl-crawl <url>` | Crawl a site following links |
| `/watercrawl-extract <url>` | Extract structured fields from a page |
| `/watercrawl-search <query>` | Search the web and return page content |
| `/watercrawl-map <url>` | Discover all URLs on a domain |
| `/watercrawl-screenshot <url>` | Take a screenshot (requires Playwright) |
| `/watercrawl-links <url>` | List all links on a page |
| `/watercrawl-batch <urls>` | Scrape multiple URLs |
| `/watercrawl-monitor <url>` | Snapshot a page and detect changes |

Or just describe what you want — Watercrawl skills activate automatically from natural language.
